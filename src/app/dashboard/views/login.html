Permissions explained
  • View your email address - We don't use your email, we just check that you have a confirmed email address.
  • Run commercials on your channel - We need this so you can start commercials from the Stahpbot dashboard.
  • Update your channel's title, game, status, and other metadata - We need this so you can update your stream title from the !title command and on the dashboard.
  • Cut VODs - We don't use this permission, it is automatically listed by Twitch for some reason. ¯\_(ツ)_/¯
  • Get a list of all subscribers to your channel - This permission is used if you enable the !subs command, it will output the current subscriber count.
  • Check if a user is subscribed to your channel - This is used for giveaways, it will check if the winner is a subscriber. This permission will be phased out soon.
