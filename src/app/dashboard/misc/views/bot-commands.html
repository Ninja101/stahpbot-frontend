Generic Commands

SyntaxDescriptionExamplePermission
!commands Outputs a link to a list of the channel's custom commands In: 
!commands
Out: 
Command List: http://stahpbot.com/commands/stahpbot

Any
!nowplaying, !song Output the current playing song from the specified source. (Available sources: Last.FM, Plug.DJ) In: 
!nowplaying
Out: 
- Now Playing: Artist - Track Name

Any
!permit, !allow [name] Temporarily allow a user to post a link without being timed out In: 
!permit Ninja101
Out: 
Ninja101 -> Ninja101 is permitted to post a link.

Mod
!quote [text] Outputs a stored quote or if text is present, will add the quoted text to the database. Only moderators can add a new quote. In: 
!quote Hello!
Out: 
Ninja101 -> Quote added!

Any
!delquote [text] Delete the quoted text from the database In: 
!delquote Hello!
Out: 
Ninja101 -> Quote removed.

Mod

Keyword Commands

SyntaxDescriptionExamplePermission
!keyword, !fkeyword, !skeyword [text] Start a keyword giveaway with the specified text. !fkeyword starts a follower-only keyword, !skeyword starts a sub-only keyword. In: 
!keyword Test
Out: 
- The keyword is 'Test', a random winner will be picked in 30 seconds.

Mod
!reroll Re-roll a winner from the previous keyword entrants. In: 
!reroll
Out: 
- Congrats: Ninja101!

Mod
!abort Aborts a running keyword giveaway. In: 
!abort
Out: 
Ninja101 -> Keyword aborted.

Mod
